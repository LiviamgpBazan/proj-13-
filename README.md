# Estrutura de Dados com Vetores em JavaScript 💚

## Sobre a atividade

Nesta atividade foram desenvolvidos programas em JavaScript utilizando vetores e estruturas de repetição.

Os exercícios tiveram como objetivo praticar:

- criação de vetores;
- armazenamento de dados;
- utilização de estruturas de repetição;
- manipulação de valores;
- lógica de programação.

## Exercícios realizados

### 1. Soma dos números pares
Programa que lê valores armazenados em um vetor e realiza a soma apenas dos números pares.

### 2. Maior valor do vetor
Programa que identifica o maior número armazenado em um vetor.

### 3. Média dos valores
Programa que calcula a média dos números armazenados no vetor.

### 4. Contagem de números positivos
Programa que percorre o vetor e conta quantos números positivos existem.
